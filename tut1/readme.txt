1. Have docker installed
2. Have a dockerID
3. run cmd: docker images or docker image ls to list your downloaded images
4. docker images have a tag that further describes the image
5. run cmd: docker pull hello-world
6. list your images
7. run cmd: docker pull hello-world:linux
8. run the image cmd: docker run -it ubuntu bash (i is interactive mode - ie interact with the container, t is the command for naming the container - makes it easy to start up again after exit)
9. Run docker ps to list all containers
10. Remember an image is the blueprint and a container is the running instance of an image.
